Kapitel 1: Einführung in C: Datentypen, Variablen & Konstanten

Der Grundstein für dein Verständnis der C-Programmierung.

1. Lernziel

Am Ende dieser Einheit bist du in der Lage, einfache C-Programme zu schreiben, die die grundlegenden Datentypen verwenden. Du kannst Variablen und Konstanten korrekt definieren, ihnen Werte zuweisen (initialisieren) und diese Werte formatiert auf der Konsole ausgeben. Du verstehst, warum Datentypen essenziell sind und wie C im Speicher mit Daten umgeht.

2. Motivation & Kontext

Warum C lernen? C ist die Grundlage vieler moderner Systeme. Betriebssysteme wie Windows, Linux und macOS sind in ihren Kernkomponenten in C geschrieben. Auch in der Welt der "Embedded Systems" – also winzigen Computern in Autos, Waschmaschinen oder medizinischen Geräten – ist C die dominierende Sprache.

Dieses Thema ist das absolute Fundament. Ohne ein solides Verständnis von Datentypen, Variablen und Konstanten ist keine sinnvolle Programmierung möglich. Alles, was du später lernst (Schleifen, Funktionen, Pointer), baut auf diesem Wissen auf. Die hier gelernten Konzepte zur Datenspeicherung sind zudem eine wichtige Vorbereitung für das Verständnis von Klassen und Objekten in C++.

3. Definitionen & Begriffe

Datentyp

Ein Datentyp legt zwei Dinge fest: einen Wertebereich (z.B. ganze Zahlen von -128 bis 127) und die darauf erlaubten Operationen (z.B. Addition, Subtraktion). C ist eine typisierte Sprache, was bedeutet, dass jede Variable genau einen Typ haben muss.

Variable

Eine Variable ist ein symbolischer Name für einen Speicherbereich im Computer. Über diesen Namen können wir Daten ablegen und wieder abrufen. In C müssen Variablen vor ihrer Verwendung definiert werden, wodurch der Name bekannt gemacht und Speicher reserviert wird. Das erstmalige Zuweisen eines Wertes nennt man Initialisierung.

Konstante

Eine Konstante ist ein Wert, der sich zur Laufzeit des Programms nicht ändern kann. Man unterscheidet:

  • Literale Konstanten: Fest im Code geschriebene Werte wie 42, 3.1415 oder "Hallo Welt".
  • Konstante Variablen (const): Variablen, die mit dem Schlüsselwort const deklariert werden und nach ihrer Initialisierung nicht mehr verändert werden können.
  • Symbolische Konstanten (#define): Werden vom Präprozessor vor der eigentlichen Kompilierung durch literale Konstanten ersetzt.

sizeof-Operator

Ein spezieller Operator in C, der die Größe eines Datentyps oder einer Variablen in Bytes zurückgibt. Dies ist extrem nützlich, um die Portabilität von Code zu gewährleisten, da die Größe von Datentypen je nach System variieren kann.

4. Schritt-für-Schritt-Erklärung

Die Grundstruktur eines C-Programms

Jedes C-Konsolenprogramm hat einen ähnlichen Aufbau:

// 1. Präprozessor-Anweisung: Bindet die Standard-Bibliothek für Input/Output ein
#include <stdio.h>

// 2. Die main-Funktion: Der Einstiegspunkt des Programms
int main(void) 
{
    // 3. Anweisungen des Programms kommen hier hinein
    
    printf("Hello World!\n"); // Beispiel: Ausgabe auf der Konsole

    // 4. Rückgabewert: 0 signalisiert dem Betriebssystem "alles ok"
    return 0;
}

Variablen definieren und initialisieren

Die Syntax lautet immer: Datentyp Variablenname; oder direkt mit Initialisierung: Datentyp Variablenname = Wert;.

// Definition ohne Initialisierung
int anzahl_aepfel; 

// Wertzuweisung nach der Definition
anzahl_aepfel = 10;

// Definition und Initialisierung in einem Schritt (bevorzugt!)
double preis_pro_apfel = 0.55; 

// Definition einer Konstante
const int TAGE_PRO_WOCHE = 7;

Die wichtigsten Basisdatentypen

C stellt mehrere grundlegende Datentypen zur Verfügung. Für den Anfang konzentrieren wir uns auf die wichtigsten, ausgehend von einem 32-Bit-System.

Datentyp Größe (in Byte) Beschreibung & Beispiel
char 1 Speichert ein einzelnes Zeichen (Buchstabe, Ziffer, Symbol). Beispiel: char buchstabe = 'A';
int 4 Der Standard-Datentyp für ganze Zahlen. Beispiel: int anzahl = -100;
float 4 Gleitkommazahl mit einfacher Genauigkeit. Beispiel: float pi_einfach = 3.1415F;
double 8 Gleitkommazahl mit doppelter Genauigkeit. Beispiel: double pi_genau = 3.14159265;

Diese Typen können mit Modifizierern wie short, long und unsigned (vorzeichenlos) kombiniert werden, um den Wertebereich anzupassen. Zum Beispiel speichert unsigned int nur positive Zahlen, kann dafür aber doppelt so hohe Werte erreichen.

Werte mit printf ausgeben

Um den Wert einer Variablen auszugeben, verwendet man Format-Platzhalter im Text. Die Funktion ersetzt diese dann durch die Werte der übergebenen Variablen.

int alter = 30;
printf("Ich bin %d Jahre alt.\n", alter); // %d ist der Platzhalter für ein int

double groesse = 1.85;
printf("Meine Größe ist %f Meter.\n", groesse); // %f ist der Platzhalter für float/double

5. Beispielprogramme

Beispiel 1: Datentypen und ihre Größen

Dieses Programm deklariert Variablen der wichtigsten Typen, initialisiert sie und gibt ihre Werte sowie ihre Größe im Speicher aus. Es kombiniert die Ideen aus Aufgabe 2 und 3 des Übungsblatts.

#include <stdio.h>

int main(void) {
    // Definition und Initialisierung verschiedener Variablen
    char initial = 'M';
    int anzahl_studenten = 150;
    unsigned int bewohner_stadt = 45000; // Für große, nur positive Zahlen
    float durchschnittsnote = 2.3F; // 'F' am Ende kennzeichnet ein float-Literal
    double pi = 3.1415926535;
    
    // Ausgabe der Werte mit passenden Format-Platzhaltern
    printf("Initial: %c\n", initial);
    printf("Anzahl Studenten: %d\n", anzahl_studenten);
    printf("Einwohnerzahl: %u\n", bewohner_stadt); // %u für unsigned int
    printf("Durchschnittsnote: %f\n", durchschnittsnote);
    printf("Pi: %f\n", pi); // %f kann auch für double verwendet werden
    printf("-----\n");

    // Ausgabe der Speichergröße mit dem sizeof-Operator
    printf("Größe von char: %zu Byte\n", sizeof(char));
    printf("Größe von int: %zu Byte\n", sizeof(int));
    printf("Größe von unsigned int: %zu Byte\n", sizeof(unsigned int));
    printf("Größe von float: %zu Byte\n", sizeof(float));
    printf("Größe von double: %zu Byte\n", sizeof(double));
    // Hinweis: %zu ist der korrekte Format-Platzhalter für das Ergebnis von sizeof.

    return 0;
}

Beispiel 2: Variable vs. Konstante

Dieses Beispiel zeigt den Unterschied im Umgang mit einer veränderbaren Variable und einer unveränderlichen Konstante.

#include <stdio.h>

int main(void) {
    // Eine konstante Variable für die Mehrwertsteuer
    const float MWST_SATZ = 0.19F;

    // Eine normale Variable für einen Nettopreis
    double netto_preis = 250.0;

    printf("Nettopreis: %.2f Euro\n", netto_preis);
    printf("Mehrwertsteuersatz: %.2f%%\n", MWST_SATZ * 100); // %% gibt ein einzelnes %-Zeichen aus

    // Berechnung durchführen
    double mwst_betrag = netto_preis * MWST_SATZ;
    double brutto_preis = netto_preis + mwst_betrag;

    printf("Bruttopreis: %.2f Euro\n", brutto_preis);
    printf("-----\n");
    
    // Der Wert einer normalen Variable kann geändert werden
    netto_preis = 300.0;
    printf("Neuer Nettopreis: %.2f Euro\n", netto_preis);

    // Der folgende Code würde einen Compiler-Fehler erzeugen!
    // Eine Konstante kann nicht geändert werden.
    // MWST_SATZ = 0.07F; // --> FEHLER: assignment of read-only variable

    return 0;
}

6. Übungsaufgaben

  1. "Hello World" in der IDE
    Richte, wie in Aufgabe 1 des Übungsblatts beschrieben, ein neues C-Projekt in deiner IDE (z.B. Visual Studio) ein. Erstelle eine `main.c` und schreibe ein Programm, das "Hello World!" auf der Konsole ausgibt.
    Hinweis / Lösungsskizze

    Das Grundgerüst mit #include <stdio.h> und int main(void) ist dein Startpunkt. Die Funktion, die du brauchst, ist printf("Dein Text hier\n");.

  2. Der Datentyp-Inspektor
    Schreibe ein Programm, das für alle in Aufgabe 2 des Übungsblatts genannten Ganzzahl- und Gleitkomma-Datentypen (char, short int, int, long int, long long int, float, double, etc.) die jeweilige Größe in Bytes auf dem Bildschirm ausgibt.
    Hinweis / Lösungsskizze

    Verwende für jeden Datentyp eine Zeile wie: printf("Größe von short int: %zu Bytes\n", sizeof(short int));. Vergiss nicht die unsigned-Varianten.

  3. Variablen-Karussell
    Definiere fünf Variablen für die folgenden Informationen: dein Alter (ganze Zahl), dein Lieblingsbuchstabe (Zeichen), die Körpergröße einer Person in Metern (Gleitkommazahl), die Einwohnerzahl deines Heimatortes (große, vorzeichenlose Ganzzahl) und den Preis eines Kaffees (Gleitkommazahl). Initialisiere sie mit passenden Werten und gib alle Werte in einem einzigen printf-Aufruf schön formatiert aus.
    Hinweis / Lösungsskizze

    Wähle die passenden Datentypen (int, char, double, unsigned long, float). Du kannst mehrere Variablen an printf übergeben: printf("Alter: %d, Buchstabe: %c ...", alter, buchstabe, ...);

  4. Der unantastbare Wert
    Definiere eine konstante Variable vom Typ int, aber weise ihr bei der Definition absichtlich keinen Wert zu. Was meldet der Compiler? Behebe den Fehler. Versuche danach, den Wert der Konstante in einer neuen Zeile zu ändern. Was passiert jetzt?
    Hinweis / Lösungsskizze

    Der Compiler wird sich beschweren, dass eine Konstante initialisiert werden muss. Nachdem du das behoben hast (z.B. const int MEINE_KONSTANTE = 100;), wird er einen weiteren Fehler melden, wenn du versuchst, ihr einen neuen Wert zuzuweisen (z.B. MEINE_KONSTANTE = 200;), da sie "read-only" ist.

  5. Mini-Taschenrechner
    Deklariere zwei Variablen vom Typ int, `a` und `b`, und initialisiere sie mit den Werten 15 und 4. Deklariere drei weitere int-Variablen: `summe`, `differenz` und `produkt`. Berechne die entsprechenden Werte und speichere sie in diesen Variablen. Gib zum Schluss alle Ergebnisse in klar beschrifteten Zeilen aus (z.B. "15 + 4 = 19").
    Hinweis / Lösungsskizze

    Die Berechnungen sind einfach: summe = a + b; etc. Für die Ausgabe kannst du die Variablen mehrfach verwenden: printf("%d + %d = %d\n", a, b, summe);.

7. Zusammenfassung & Spickzettel

// --- GRUNDGERÜST ---
#include <stdio.h>

int main(void) {
    // Dein Code hier
    return 0;
}

// --- VARIABLEN & KONSTANTEN ---
// Definition         | Initialisierung
int anzahl;           | anzahl = 10;
char zeichen = 'A';   // Definition & Initialisierung
const double PI = 3.14159; // Konstante

// --- WICHTIGE DATENTYPEN ---
// Ganze Zahlen
char          // 1 Byte
short int     // 2 Bytes
int           // 4 Bytes
long long int // 8 Bytes
// Füge 'unsigned' hinzu für nur-positive Werte (z.B. unsigned int)

// Gleitkommazahlen
float   // 4 Bytes, mit F am Ende: 2.5F
double  // 8 Bytes

// --- GRÖSSE ERMITTELN ---
size_t groesse = sizeof(int); // ergibt 4 (auf 32-Bit System)

// --- AUSGABE MIT PRINTF ---
printf("Text...");
// Format-Platzhalter:
// %d : int (signed)
// %u : unsigned int
// %c : char
// %f : float / double
// %zu: size_t (für sizeof)
// \n : Zeilenumbruch
// %% : Ein %-Zeichen ausgeben
